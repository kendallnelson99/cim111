Overwhelmed - Representing Emotion



Objective

The goal of this project was to represent an emotion of our choosing. I chose overwhelmed. I created a paper box with a silhouette of a person and cut out shapes and icons that I thought visibly showed the emotion.>

Final video

Sculpture

I got the template to make this box using Template Maker. I got the silhouette outline from google. I used Cricut Maker to cut the box and it also drew the silhouette

Code

I used a sequencing code to create my pattern. It goes Red, Yellow, Red. It creates kind of an orange color when the yellow is introduced.

Circut

3- 220 ohm resistors. 1- 10K resistor

2 - Red LED 1- Yellow LED

1 - Ardiuno uno

1 - Switch

Issues

Because of my sequencing pattern it was difficult for me to get my push button code to work. Another think is i should have put a Duralar paper to diffuse the light and so that you cant see the LEDs from the outside.
